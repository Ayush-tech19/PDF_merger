📄 PDF Merger (Python)

A simple Python script to merge multiple PDF files into a single PDF using PyPDF2.

-- Features

- Merge multiple PDFs into one
- Easy to use (CLI based)
- Lightweight and fast


-- Requirements

- Python 3.x
- PyPDF2 library

Install dependency:

pip install PyPDF2

-- How to Use

1. Place all your PDF files in the same folder as the script.

2. Run the script:

python main.py

3. Enter number of PDFs you want to merge.

4. Enter file names one by one (example: file1.pdf, file2.pdf)

5. Output file will be created:

merged-pdf.pdf

---

--
Example

Input:

How many pdfs do you want to merge?
2
Enter the name of pdf 1: a.pdf
Enter the name of pdf 2: b.pdf

Output:

merged-pdf.pdf

---

⚠️ Notes

- Make sure all PDF files exist in the same directory
- File names should be correct (case-sensitive)
